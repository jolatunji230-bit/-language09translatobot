# 🤖 Language09 Translator Bot

A powerful Telegram bot that translates text between 100+ languages instantly.

## 🌟 Features

- 🌍 **100+ Languages** - Support for major world languages
- ⚡ **Instant Translation** - Quick and accurate translations
- 🔍 **Auto Detection** - Automatically detects source language
- 🎯 **Target Language** - Set your preferred target language
- 📊 **Statistics** - Track your usage
- 🎨 **Inline Buttons** - Easy-to-use interface
- 🔒 **Privacy** - No data stored

## 🚀 Commands

| Command | Description |
|---------|-------------|
| `/start` | Show welcome message |
| `/help` | Display help information |
| `/translate [text]` | Translate specific text |
| `/lang [code]` | Set target language |
| `/detect [text]` | Detect language of text |
| `/languages` | List all supported languages |
| `/stats` | Show your statistics |
| `/about` | About this bot |

## 📦 Deployment

### 1. Create Bot on Telegram
- Search for `@BotFather` on Telegram
- Send `/newbot` and follow instructions
- Copy your bot token
