# 🤖 Language09 Translator Bot

Telegram bot that translates text between 100+ languages.

## Commands
- `/start` - Welcome
- `/help` - Help
- `/translate <text>` - Translate
- `/lang <code>` - Set language
- `/detect <text>` - Detect language
- `/languages` - List all languages
- `/about` - About

## Deployment
1. Create bot with @BotFather
2. Get BOT_TOKEN
3. Deploy on Railway
4. Add BOT_TOKEN as environment variable

## Technologies
- Python 3
- python-telegram-bot
- googletrans
- Railway
