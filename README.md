# Language09 Translator Bot

A Telegram bot that translates text between 30+ languages.

## Commands
- `/start` - Welcome
- `/help` - Help
- `/lang [code]` - Set language
- `/translate [text]` - Translate
- `/languages` - List all

## Deploy on Railway
1. Create bot with @BotFather
2. Get BOT_TOKEN
3. Deploy on Railway
4. Add BOT_TOKEN as environment variable
